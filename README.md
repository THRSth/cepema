
# Pré-requisitos

 - [Node](https://nodejs.org/en/download/)
 - [Yarn](https://classic.yarnpkg.com/en/docs/install#debian-stable)

# Passo a passo

### 1. clonar o repositório

```
git clone https://github.com/THRSth/cepema.git
```

### 2. Acessar pasta e baixar dependências

```
cd cepema
yarn install
```

### 3. Iniciar aplicação

```
yarn start
```
